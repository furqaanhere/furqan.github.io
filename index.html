Projects

Apple Grading System (AGS)

Problem: Manual grading is inconsistent, slow, and labor-intensive.

Solution: Automated grading using CNN and computer vision.

System Architecture:
Input → Image Capture → Preprocessing → CNN → Classification → Sorting

CNN Model:
Uses convolution layers to detect patterns like color, defects, and texture. Pooling layers reduce data while preserving features, and dense layers classify apples into quality categories.

Impact:
Improves accuracy, reduces human effort, and enables scalable agricultural automation.
